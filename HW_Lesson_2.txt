1. cat --> считывает данные из файла и выводит их.

   1.1. создайте два файла с данными, а затем объедините их в один.
   		cat >> text_file_1.txt (перенаправляем поток и сохраняем в файл, вводим текст, в конце ctrl+C)
   		cat >> text_file_2.txt (аналогично делаем для второго файла)
   		cat text_file_1.txt text_file_2.txt >> text_file_3.txt (открываем оба файла, перенаправляем поток и сохраняем в файл, получается объединение содержимого файлов)

   1.2. просмотрите содержимое созданного файла
   		cat text_file_3.txt

   1.3. скопируйте файл, дав ему новое имя
   		cat text_file_3.txt > text_file_3_new.txt

2. Напишите команду, которая будет показывать количество скрытых файлов в домашнем каталоге пользователя.
   
   find ~ -mindepth 1 -prune -name '.*' | wc -l >> ans_2.txt

   		find 			--> сейчас будем искать
   		~ 				--> в домашнем каталоге пользователя
   		-mindepth 1		--> ограничивая глубину поиска до одного символа
   		-prune 			--> не спускаясь в подкаталоги
   		-name			--> файл с названием
   		'.*'			--> начинающимся на . 

   		(благодаря -mindepth 1 избегаем случая '..')

   		|				--> перенаправляем результат, полученный выше на
   		wc				--> подсчет количества
   		-l				--> строк в нем
   		>> ans_2.tex    --> получившийся ответ записываем в файл

3. Создаем свой веб калькулятор. 

   3.1. Используя команду curl загрузить и сохранить на Ubuntu скрипт по ссылке
		https://raw.githubusercontent.com/mtuktarov/linux_goodies/master/docker/python/uwsgi/flaskapp.py

		sudo apt install curl 
		(скачиваем curl)

		curl https://raw.githubusercontent.com/mtuktarov/linux_goodies/master/docker/python/uwsgi/flaskapp.py 
		(смотрим на файл)

		curl -o ~/'Рабочий стол'/flaskapp.py https://raw.githubusercontent.com/mtuktarov/linux_goodies/master/docker/python/uwsgi/flaskapp.py 
		(url -o --> скачиваем файл  ~/'Рабочий стол'/flaskapp.py --> на рабочий стол с названием flaskapp.py)

   3.2. Попытаться запустить скрипт, используя полный путь (абсолютный) или относительный (./<script_name>), добавив права на выполнение при необходимости

        sudo ls -la ~/'Рабочий стол'/flaskapp.py
        (смотрим какие есть права)

        sudo chmod +x ~/'Рабочий стол'/flaskapp.py
        (добавляем файлу права на выполнение)

   3.3. Установить питон2 в убунту.

   		sudo apt install python  

   3.4. При необходимости установите отсутствующие модули, используя программу pip.

   		pip3 install Flask (не удалось установить pip для python v2, но через pip3 в целом тоже работает)

   3.5. Просмотрите код программы и найдите, какая переменная окружения используется для указания адреса, на котором программа слушает входящие запросы. 
        Поменяйте адрес по умолчанию на тот IP адрес ubuntu, который доступен с хоста и убедитесь, что можете открыть программу со стороны хоста. 
        Код самой python программы менять не нужно

        host = os.getenv('FLASK_HOST', '127.0.0.1')  --> хост
    	  port = os.getenv('FLASK_PORT', 5000)		 --> порт

        export FLASK_HOST = ip компа (192.168.1.8)

   3.6. Сделайте так, чтобы калькулятор можно было запустить из любого рабочего каталога командной строки, используя одно только название. Настройка должна быть устойчива к перезагрузке.

   		sudo mkdir /opt/bin/			--> создаем папку /opt/bin/
   		sudo mv ~/'Рабочий стол'/flaskapp.py /opt/bin/	--> перемещаем скрипт с рабочего стола в папку /opt/bin/
   		export PATH=/opt/bin:$PATH		--> добавляем путь к папке /opt/bin/ в существующую
